# Product Overview

Linux From Scratch (LFS) Automated Build System - A platform for building Linux From Scratch (LFS 12.0) with an interactive learning frontend.

## Purpose

- Teach users how to build a complete Linux system from source code
- Provide step-by-step guided lessons for LFS compilation
- Track user progress through modules and lessons
- Offer an AI chat assistant for help during the build process

## Live Site

https://lfs-by-sam.netlify.app

## Key Features

- Interactive learning modules covering environment setup, source downloads, toolchain building, and kernel compilation
- Command execution guidance with copy-paste ready shell commands
- User authentication and progress tracking via Firebase
- Admin dashboard for monitoring user activity
- AI-powered chat assistance using Google Vertex AI
- Build status monitoring and log viewing

## Target Users

- Linux enthusiasts wanting to understand system internals
- Developers learning low-level system programming
- Students studying operating systems
