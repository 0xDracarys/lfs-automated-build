## Instructions for Creating Tables:

1. **Format**: Use Word's built-in table styles (Light Shading - Accent 1)
2. **Header Row**: Bold text, shaded background
3. **Alignment**: Text left-aligned, numbers right-aligned
4. **Captions**: Format as "Table X. [Description]" above table in 10pt italic
5. **Borders**: Use simple gridlines, avoid heavy borders
6. **Font Size**: 10pt for table content, 9pt acceptable for dense data
7. **Width**: Tables should fit within page margins (no manual page breaks)
