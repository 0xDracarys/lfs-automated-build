# 07. User Guide

This section provides step-by-step instructions for users interacting with the LFS Automated Build platform. The UI is designed for clarity and accessibility (World Wide Web Consortium, 2021).

## Getting Started

1. Register or log in using Firebase Auth (see Introduction Objective 3).
2. Navigate to the Install Wizard and submit a build (see Figure 21).
3. Monitor build progress and logs in real time via the dashboard and log viewer.
4. Download completed artifacts using the provided link (see Figure 22).

## Troubleshooting

- Common issues (e.g., build failures, network errors) are surfaced in the log viewer with actionable messages.
- For persistent errors, consult the FAQ or contact support.

## Accessibility

- The platform follows WCAG 2.2 guidelines for color contrast, keyboard navigation, and screen reader support (World Wide Web Consortium, 2021).
