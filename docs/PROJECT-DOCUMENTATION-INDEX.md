## üîß Development

### Quick Start
```bash
# Clone repository
git clone [repository-url]
cd lfs-automated/lfs-learning-platform

# Install dependencies
npm install

# Set up environment
cp .env.example .env.local
# Add Firebase credentials

# Start development
npm run dev

# Open browser
http://localhost:3000
```

### Deployment
```bash
# Deploy to Netlify
cd lfs-learning-platform
netlify deploy --prod
```
